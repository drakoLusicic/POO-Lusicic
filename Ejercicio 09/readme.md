# 🎨 Ejercicio 09 - Lusicic 🖥️

¡Bienvenido a **Ejercicio 09 - Lusicic**! Este es un proyecto de aplicación de escritorio desarrollado en **Qt** con **C++**, que incluye un sistema de login, una ventana principal y funcionalidades de pintura. Utiliza una base de datos **SQLite** para gestionar datos. 🚀

## ✨ Características

- 🔐 **Sistema de Login**: Autenticación segura con interfaz gráfica.
- 🖼️ **Ventana Principal**: Interfaz principal de la aplicación.
- 🎨 **Módulo de Pintura**: Herramientas para dibujar y crear arte digital.
- 💾 **Base de Datos SQLite**: Almacenamiento persistente de datos.
- 🛠️ **Qt Framework**: Desarrollo moderno con widgets y señales/slots.

## 📋 Requisitos

- 🖥️ **Qt Creator** o **Qt SDK** (versión 6.x recomendada)
- 🛠️ **MinGW** o **MSVC** compiler
- 💻 **Windows** (o compatible con Qt)
- 📚 **SQLite** (incluido en Qt)

## 🚀 Instalación y Compilación

1. 📥 **Clona o descarga** el proyecto en tu máquina local.
2. 🖥️ **Abre** el archivo `Ejercicio09-Lusicic.pro` en **Qt Creator**.
3. 🔧 **Configura** el kit de compilación (Desktop Qt 6.x MinGW 64-bit recomendado).
4. ▶️ **Compila** el proyecto usando el botón "Build" o `Ctrl+B`.
5. 🎯 **Ejecuta** la aplicación con `Ctrl+R` o el botón "Run".

### 📁 Estructura del Proyecto

```
Ejercicio09-Lusicic/
├── Ejercicio09-Lusicic.pro          # Archivo de proyecto Qt
├── main.cpp                         # Punto de entrada
├── mainwindow.cpp / .h / .ui        # Ventana principal
├── logindialog.cpp / .h / .ui       # Diálogo de login
├── pintura.cpp / .h                 # Módulo de pintura
├── ejercicio09.sqbpro               # Base de datos SQLite
├── instructions_sqlite.txt          # Instrucciones para SQLite
└── build/                           # Archivos de compilación
```

## 🎮 Uso

1. 🔓 **Inicia sesión** en la aplicación usando el diálogo de login.
2. 🖼️ **Navega** por la ventana principal para acceder a las funciones.
3. 🎨 **Usa el módulo de pintura** para crear y editar dibujos.
4. 💾 **Guarda** tus trabajos en la base de datos integrada.

## 🐛 Solución de Problemas

- ⚠️ Si hay errores de compilación, verifica que Qt esté correctamente instalado.
- 🔍 Consulta `instructions_sqlite.txt` para detalles sobre la configuración de la base de datos.
- 📞 Para más ayuda, revisa la documentación de Qt o contacta al desarrollador.
